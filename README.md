# MidScope — AI Lab Report Guide

Full-stack medical diagnostics application with AI-powered lab report analysis.

## Stack
- **Backend**: FastAPI + Python (server.py)
- **Frontend**: React 19 + Tailwind CSS + Shadcn/UI
- **Database**: Supabase (Postgres + RLS)
- **AI**: Emergent Integrations LLM

## Features
- Lab report analysis with clinical reference ranges
- Multi-language support (70+ languages)
- Health profile with age/sex-specific ranges
- PWA support
- Subscription tiers (Free/Premium)

## Repository
- `backend/` — FastAPI server, health profile logic, lab reference DB
- `frontend/` — React SPA with Shadcn components
- `scripts/` — Utility scripts
- `tests/` — Test suite
